## Email Triage Assistant

An AI-powered tool to manage and streamline email communication by summarizing threads, categorizing messages, and suggesting professional replies.

# Technologies Used

--Python

--OpenAI GPT API

--pandas, numpy, nltk

# Setup

1. Clone the repository:
   
      git clone https://github.com/Ranga29/email-triage-assistant.git

2. Install dependencies:
   
      pip install -r requirements.txt

3. Open src/main.py and set your OpenAI API key.

4. Run the script:

      python src/main.py
